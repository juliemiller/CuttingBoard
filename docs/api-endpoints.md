# API Endpoints

## HTML API

### Users

- `GET /users/new`
- `POST /users`
- `PATCH /users`

### Session

- `GET /session/new`
- `POST /session`
- `DELETE /session`

## JSON API

### Recipes

- `GET /api/recipes`
  - Boards index/search
- `POST /api/recipes`
- `GET /api/recipes/:id`
- `PATCH /api/recipes/:id`

### Boards

- `GET /api/boards`
- `POST /api/boards`
- `GET /api/boards/:id`
- `PATCH /api/boards/:id`
- `DELETE /api/boards/:id` 


### Categories

- `GET /api/categories`
